# Audie 0.1.1 Preview

## English

Audie's first downloadable macOS preview.

### What's new

- Start voice input with Fn or a custom shortcut
- Insert streaming speech recognition results into the active app
- Optionally enhance text with AI
- Guided microphone, Accessibility, and Input Monitoring setup

### Installation

Download the Apple Silicon DMG, drag Audie into Applications, then run:

```bash
xattr -dr com.apple.quarantine /Applications/Audie.app
open /Applications/Audie.app
```

This preview is not notarized by Apple. Updating may require granting permissions again.

## 中文

Audie 首个可下载的 macOS 预览版本。

### 新功能

- 使用 Fn 或自定义快捷键开始语音输入
- 将流式语音识别结果自动输入当前应用
- 支持可选的 AI 文本润色
- 提供麦克风、辅助功能和输入监控授权引导

### 安装

下载 Apple Silicon DMG，将 Audie 拖入“应用程序”，然后运行：

```bash
xattr -dr com.apple.quarantine /Applications/Audie.app
open /Applications/Audie.app
```

当前预览版未经 Apple 公证；更新后可能需要重新授予系统权限。
